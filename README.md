# KawaiiKings
Who is the cutest animal?

A game hosted on the Ethereum framework in which users upload, vote and bid on the cutest animal.  

Users:
-Metamask 
-Eth
-Low GWEI
-ID = Public Key

Vote:
-Sol closed contract
-blind votes

Bids: 
-blind bids

Payout:
-top animals (uploader 10% bidders 90%)
-progressive scaling
